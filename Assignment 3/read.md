# 🎓 Student Scoreboard (React Project)

## 📌 Overview

This project is a **Student Scoreboard Application** built using React.
It allows users to manage student records, update scores dynamically, and view pass/fail status with a modern UI.

---

## 🚀 Features

* ✅ Add new students with name and score
* ✅ Update student scores in real-time
* ✅ Automatically display **Pass/Fail status**

  * Pass → Score ≥ 40
  * Fail → Score < 40
* ✅ View total students, passed students, and average score
* ✅ Clean and responsive UI with a futuristic theme

---
